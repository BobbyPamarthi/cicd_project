# Simple Node.js Website

A clean, responsive website built with **zero dependencies** — just Node.js core modules.

## Project Structure

```
nodejs-website/
├── server.js          # HTTP server
└── public/
    ├── index.html     # Main page
    ├── style.css      # Styles
    └── app.js         # Client-side JS
```

## Getting Started

1. Make sure [Node.js](https://nodejs.org) is installed
2. Run the server:

```bash
node server.js
```

3. Open your browser at [http://localhost:3000](http://localhost:3000)

## No npm install needed!
Uses only built-in Node.js modules: `http`, `fs`, and `path`.
